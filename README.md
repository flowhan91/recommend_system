# recommend_system

# 추천기능 개발 프로젝트

사용자 리뷰, 태그, 메타데이터를 기반으로 개인화 추천 기능을 개발한 프로젝트입니다.  
본 프로젝트는 텍스트 데이터 분석과 유사도 계산을 통해 사용자에게 적합한 장소/콘텐츠를 추천하는 것을 목표로 합니다.

## 1. 프로젝트 개요

기존의 단순 인기순 추천 방식은 사용자의 취향과 맥락을 충분히 반영하지 못한다는 한계가 있습니다.  
이를 개선하기 위해 본 프로젝트에서는 리뷰 데이터, 카테고리 정보, 태그, 위치 정보 등을 활용하여 **개인화 추천 기능**을 구현했습니다.

### 목표
- 사용자 선호를 반영한 추천 기능 개발
- 리뷰/태그 기반 콘텐츠 임베딩 및 유사도 추천 구현
- 추천 결과의 활용 가능성 검토

---

## 2. 주요 기능

- 리뷰 텍스트 전처리
- 장소/콘텐츠별 메타데이터 정리
- 임베딩 기반 유사도 계산
- 사용자 선호 태그 기반 추천
- 조건 필터링(예: 카테고리, 위치, 분위기 등)
- 추천 결과 상위 N개 반환

## 4. 추천 시스템 설계 방식

본 프로젝트의 추천 기능은 다음 흐름으로 동작합니다.

1. 사용자 리뷰 및 장소 데이터를 수집
2. 텍스트 전처리 및 핵심 키워드 추출
3. 장소별 특징 벡터(임베딩 또는 태그 벡터) 생성
4. 사용자 선호 정보와 장소 벡터 간 유사도 계산
5. 필터 조건을 반영하여 최종 추천 결과 반환

### 추천 로직 예시
- 사용자가 선호하는 태그와 유사한 장소 추천
- 이전에 좋아한 장소와 유사한 장소 추천
- 지역/카테고리 필터를 적용한 맞춤형 추천
